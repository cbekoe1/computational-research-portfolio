# Computational Research Portfolio

This repository contains a collection of 12 mid‑level computational research projects across four domains:

- Wind Farm Simulation
- Infectious Disease Modeling
- Climate & Energy Systems
- Numerical Optimization

Each project includes:
- Modular Python code (`src/`)
- A Jupyter Notebook for analysis (`notebooks/`)
- Input data (`data/`)
- Generated results (`results/`)
- A research‑grade PDF report (`report/`)
- A project‑level README

All work is done using Python, SciPy, NumPy, Pandas, Matplotlib, and Jupyter Notebook.
